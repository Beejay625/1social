# 1Social Dashboard

A social media management dashboard built with Next.js. Compose once and share to multiple platforms (Farcaster, Instagram, X, Lens, Mirror).

## Quick Start

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Features

- Multi-platform content publishing
- Scheduling & workflow automation
- Analytics & reporting
- AI content generation
- Team collaboration
- API management & webhooks

## Tech Stack

- **Next.js** (App Router)
- **Tailwind CSS**
- **Reown AppKit** + **Wagmi** + **Viem** (wallet connection)
- **TypeScript**

## Environment

Set `NEXT_PUBLIC_PROJECT_ID` from your [Reown dashboard](https://dashboard.reown.com).
